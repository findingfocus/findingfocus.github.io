Joust

My remake of the 1982 Williams Electronics game, Joust. Built in Love2d with Lua. My version comes with 25 waves of enemies, a two-player mode, and a leaderboard!

Tashio Tempo

Tashio Tempo is my new game inspired by Link's Awakening on Game Boy. It is a top-down game filled with adventure, magic, and music. I released the current version of the game featuring gecko entities, spell-casting abilities, and several screens to explore!

Weather

This is my submission to the 2022 OneLoneCoder Code Jam. Click below to play a small demo, or download the executable from the GitHub release page to play to full version with interactive sound effects!

Tap the very edges of the game's canvas to fly

Desktop Instructions:

Tak: A Beautiful Game

Tak is a board game from the novel "The Wise Man's Fear", created by James Ernest and Patrick Rothfuss. I programmed my own version of the game in Love2d and Lua. It is a competitive board game for two players, so grab a friend and let the game commence!
